# varene

## 简介

一个更加轻便，易用，可快速移植在其他php程序中的Telegram Bot。开发的目的是为了减少对其他包的依赖，以便能在最简单的环境中运行；使用更加简单的代码，快速搭建起属于自己的机器人。

## 依赖

1. php >= 7.0.0
2. redis 
3. https支持
